作品名：InWater〜水の中にいる〜
ジャンル：シューティング
開発環境：c++,DirectX9
動作環境：Windows7
制作期間：3週間


-----説明-----
DirectX9で製作しました。
画面内にいる敵をできるだけ多く倒すゲームです。
私自身が水の中を表現したかったので制作しました。

-----見てほしいところ-----
点の描画を使い泡を表現しました。
泡の動きに乱数を使い不規則な動きをさせることで、より泡を表現できるように工夫いたしました。
攻撃があたったときにゼリーのような質感をだしました。

-----制作して感じたこと-----
今まで学んだことを生かしながら制作したので、すんなりと自分の思うような処理が追加していけました。
しかし、まだ技術力不足を感じるときもあったので、今後もさらに勉強をして次の制作に活かせれるように頑張りたいと思います。
今回新たに学べた知識や技術があるので、それらも活かしていきたいと思います。

-----今後追加していく要素-----
・スコア表示
・敵の種類を増加
・敵のAIを作成
・シェーダーを使って海面と海底の表現

-----操作方法-----
移動
W：前進
A：左に平行移動
S：後退
D：右に平行移動

視点変更
マウス：左右に360°回転　上下には制限がある

攻撃
左クリック：通常攻撃の泡
右クリック：下にいる敵への爆弾
スペース：必殺技の泡
